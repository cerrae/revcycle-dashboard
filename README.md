## 📋 Project Brief

| | |
|---|---|
| **Problem** | A/R aging blind spots and high claim denial rates with no centralized visibility across billing, providers, and insurance teams |
| **Solution** | Real-time dashboard surfacing aging buckets by risk level, top denial reasons, payer mix, and provider collection rates |
| **Impact** | 15% A/R reduction achieved through similar process improvements at Harlow Dental (Apr 2024–Present) |
| **Methodology** | Revenue cycle management · KPI tracking · Cross-functional stakeholder reporting |

---

## ✨ Features

- **KPI Cards** — At-a-glance metrics: A/R collected, outstanding balance, claim denial rate, and average days in A/R
- **A/R Aging Table** — Claims broken into 0–30 / 31–60 / 61–90 / 91–120 / 120+ day buckets with risk indicators (Low / Med / High)
- **Payer Mix Donut Chart** — SVG-rendered breakdown of collections by insurance payer
- **Denial Reason Analysis** — Horizontal bar chart of top 5 denial causes (missing auth, eligibility, duplicates, code mismatch, timely filing)
- **Provider Performance** — Sparkline chart tracking 6-month collection rate trends per provider
- **Period Switching** — Toggle between 30-day, 90-day, and YTD data views
- **Live Clock** — Real-time timestamp in the header
